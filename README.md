
# Einstein's Quest

Game for COSI event.

## Engine and Build

* The game is devloped in Unity version `2021.3.18f1`.

* The game is built for web (HTML5/OpenGL).

* The game includes 2 additional packages: `Input System` (1.4.1) and 
  `Universal RP` (12.1.10).
